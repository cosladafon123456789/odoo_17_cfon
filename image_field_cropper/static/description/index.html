This module allows users to select a portion of an image and crop it according to their desired dimensions.

Installation

- Copy image_field_cropper module to addons folder

- Install the module normally like other modules

1. Usage

You will get a new icon for Open Image Crop on Binary Image Field.



2. Open Image in Croppper

Click on the Crop Icon, Selected image will open in Crop Editor


3. Crop Your Image
