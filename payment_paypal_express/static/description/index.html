Why Paypal Payment Acquirer?

Because it is one of the most secure and reliable payment methods, PayPal intelligently presents the most relevant payment types to your shoppers, automatically. It makes payment easier for customers to complete their purchases using methods like Pay with Venmo, PayPal Credit, Credit Card payments, and other payment types.

Track Transaction History

The Odoo admin can access order transaction details in the backend, including payment mode details, and the order form, which displays the customer's payment method. Transactions are also stored in the PayPal merchant portal of the admin.

Disabling Funding

This feature allows the admin to disable any payment option such as card, pay later, etc. By this the admin can accept payment only in their suitable options. Admin only have to enter the option of payment in disabling funding setting in backend.

Technical Requirements For Payment Paypal Express Payment Gateway
